== Description ==

Adds toggle search form on you site using this awesome Beaver Builder module.

== Installation ==

1. First you will purchase the **Toggle Search Form** plugin.
2. Login to WP Beaver World site (https://www.wpbeaverworld.com/my-account/) and download the pack from **Download** page. 
3. Now login to your site's Dashboard and go to Plugins->Add New page. Click on upload link, browse the zip file and then hit on install now button.
4. Click on activate link and activate the plugin.

== Frequently Asked Questions ==

1. How do you activate the license key?

- a. Login to WP Beaver World site (https://www.wpbeaverworld.com/my-account/), click on **License Key** link and get the key. 
- b. Login to your site Dashboard and navigate to Settings -> TSF License Key
- c. Enter the license key into the text field and hit on "Activate License" button
